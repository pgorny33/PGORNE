# P.GÓRNE temporary homepage placeholder

This is a lightweight static placeholder homepage for pgorne.com.

## What is included
- `index.html`
- `assets/pgorne-logo-animation-1080.mp4` — compressed web version of the intro animation
- `assets/pgorne-logo-animation-720.mp4` — smaller mobile fallback
- `assets/poster.jpg` — fallback poster image

## File size
Original uploaded animation: about 42.7 MB.
Compressed 1080 web animation: under 1 MB.
This is safe to upload through GitHub's browser uploader.

## Deploy with GitHub + Vercel
1. Create a new GitHub repository.
2. Upload the contents of this folder, not the ZIP itself.
3. In Vercel, choose New Project and import the GitHub repository.
4. Framework preset: Other.
5. Build command: leave blank.
6. Output directory: leave blank.
7. Deploy.
8. In Vercel, go to Project Settings > Domains and add `pgorne.com` and `www.pgorne.com`.
9. Vercel will give you DNS records.
10. In GoDaddy DNS, replace the Squarespace records with the Vercel records.

Recommendation: keep the domain registered at GoDaddy for now. Only change DNS. Do not transfer the domain yet.
